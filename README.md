# Zeroth Principles

> **Тип репо:** Foundation (единственный в своём роде — стоит над типологией)
> **Source-of-truth:** Да (для нулевых принципов и карты иерархии принципов)

---

## Что такое нулевые принципы

Нулевые принципы — самые общие транс-дисциплинарные ограничения, которые отсекают неправильные модели и делают рассуждения проверяемыми. Они работают **в любой области**: в инженерии, в науке, в бизнесе, в творчестве.

Нулевые принципы **дисциплинируют** все остальные уровни: первые, вторые, третьи. Кто их освоил — тот не путает корреляцию с причинностью, не строит моделей, которые рассыпаются при проверке, и не принимает решений на основе наитий.

> Источник: Левенчук А.И. «Нулевые, первые, вторые и третьи принципы» (2026-02-17)

## Как пользоваться

1. **Новый человек в экосистеме** — начни с этого файла, затем [hierarchy.md](hierarchy.md) для понимания карты уровней.
2. **Проверка решения** — пройди по 6 принципам: не нарушаю ли? Каждый файл в [principles/](principles/) содержит типичные нарушения.
3. **Спор о подходе** — вернись к нулевым принципам: какой из ZP.1–ZP.6 нарушается?
